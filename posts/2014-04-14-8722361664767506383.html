Compile htop command for Android [Arm Cross Compile]

April 14, 2014

My environment :

  • Ubuntu 12.04
    - arm-linux-gnueabi-gcc
  • git
  • make
  1. htop need ncurses library for compile. i will compile ncurses.

wget http://ftp.gnu.org/pub/gnu/ncurses/ncurses-5.9.tar.gz
tar xvfz ncurses-5.9.tar.gz
cd ncurses-5.9/
./configure --host=arm-linux-gnueabi --prefix=/tmp/ncurses
make -j8
make install

  1. Download & Compile htop

wget http://hisham.hm/htop/releases/1.0.2/htop-1.0.2.tar.gz
tar xvfz htop-1.0.2.tar.gz
cd  htop-1.0.2
./configure --host=arm-linux-gnueabi --prefix=/tmp/htop_bin LDFLAGS="-L/tmp/ncurses/lib" CFLAGS="-static" --disable-unicode
make -j8
make install

  1. You can found htop binary in /tmp/htop_bin/bin/

cd /tmp/htop_bin/bin/
