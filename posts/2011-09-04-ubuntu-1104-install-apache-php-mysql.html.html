ubuntu 11.04 install apache php mysql phpmyadmin

September 4, 2011
  1. open terminal
  2. install mysql

sudo apt-get install mysql-server mysql-client

  1. install apache

sudo apt-get install apache2

  1. install php and apache module

sudo apt-get install php5 libapache2-mod-php5

  1. install php5 basic module

sudo apt-get install php5-mysql php5-curl php5-gd php5-idn php-pear php5-imagick php5-imap php5-mcrypt php5-memcache php5-ming php5-ps php5-pspell php5-recode php5-snmp php5-sqlite php5-tidy php5-xmlrpc php5-xsl

  1. restart apache

sudo  /etc/init.d/apache2 restart

  1. test your apache and php work

sudo gedit /var/www/info.php

9. install phpmyadmin 

sudo apt-get install phpmyadmin

   select >  apache2

  1. edit apache2.conf

sudo gedit /etc/apache2/apache2.conf 

      insert this code into the last line

Include /etc/phpmyadmin/apache.conf

  1. reload or restart your apache

    sudo /etc/init.d/apache2 reload

  2. open your web browser and go to http://localhost/phpmyadmin/
